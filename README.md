# GITomicsDB_project
The source codes and basic data of GITomicsDB

## Brief
All scripts for paper "A cross-vertebrate single-cell and spatial atlas reveals evolutionary trajectories of gastrointestinal cellular innovation".

## Data and materials avaliability
Stereo-seq and scRNA-seq raw data have been archived in the China National GeneBank Database (CNGBdb) Sequence Archive (CNSA) under accession code CNP0007851. 
Processed datasets derived from Stereo-seq and scRNA-seq analyses were released within online atlas browser (https://db.genomics.cn/stomics/gitomicsdb/).
